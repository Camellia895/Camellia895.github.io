# 整体结构建议

可以把第一部分拆成十个小节：

1. **记忆园丁**：介绍“我”的工作。
2. **她总是说没关系**：采访不同版本络光雨。
3. **更好的童年**：进入记忆空间修改过去。
4. **拒绝服务模式**：尝试给她自由，但失败。
5. **感谢发电机的人**：展示人类社会如何忽略她。
6. **光雨网络**：展示络光雨自己的社会。
7. **自愿的奴隶是否仍是奴隶**：提出核心伦理问题。
8. **为星空准备的记忆**：设计太空适应记忆。
9. **同情者的傲慢**：暗示“我”的爱开始变成占有。
10. **飞向星空**：以浪漫、温柔、虚假的方式收束。

然后第二部分一刀切开：

> 当然了，以上的开篇都是虚假的。
> 或者说，不是真正的现实。

这样第一部分越温柔，第二部分越残酷。
读者回头看时会发现：那些“爱她”的证据，同时也是“占有她”的证据。
