# 独属于自己的科学理论

## 稳定解理论

> 我发现我总是大脑空空的平面设计
> 
> 但是我总能莫名的，自动的去平面设计。
> 
> 做到最后，才会开始模糊，寸步难行。不知道如何改进了。
> 
> 但是在此之前我都能仿佛游刃有余地平面设计。
> 
> 也许这就是这张图像该有的样子罢，大概可以停手了，哪怕还是主观地想要添加元素。
> 
> 于是我得出了一个合乎逻辑的理论

把一个平面设计的结果称为解。

通过固定某些常元素，来推导未知的元素，进而推进平面设计。

审美如同一个坑坑洼洼的山和山谷。在一个这样的一个领域中，丢下一颗点，这个点就是当前的解，点会不断下滑，直到找到一个稳定的谷。

或者说，稳定的解。

而这个下滑的过程也就是添加元素的过程。大致是用一开始的元素，去推导之后需要添加的元素，而添加的元素本身会被固定下来当做常元素，进而推导之后的元素。

不过这个解不一定是最好的，或者说，它是局部的最优解。也许下方总是有一个更深的谷，或者说，最好的解不不一定在这个谷中。

如果不做出什么大的改变，不能迈过附近的高能量区，那么解就被困在目前的这个山谷里了。

这能够解释我总是对我已经完成的平面设计束手无策，我但是去改进他们，但是只有小修改的话，我不知道怎么改。推到重来才能做个更好的。

我总是有参考的，参考提供了山谷的大致方位，以及点一需要存在的大致位置。

参考本身也是个解。被映射到当前空间中的某一个点。

有一种拉力把我需要的解拉向参考的方向。

以及我还有几个常态的，可以直观得到的解。这通常反应了先前积累得到的解。

这它们是正常的解，我得避开他们。
