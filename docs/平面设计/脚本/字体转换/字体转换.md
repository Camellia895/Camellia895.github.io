# 字体去重与格式转换

本工具可以上传字体文件/文件夹，自动识别重复的字体（基于元数据匹配），支持在 WOFF2 / WOFF / SFNT / TTF 格式之间批量转换并打包下载。

## 使用说明

1. **上传字体**：拖放字体文件或文件夹到上传区，支持 .ttf、.otf、.sfnt、.woff、.woff2
2. **自动去重**：工具会自动读取字体元数据（字体名称、样式、字重），将相同的字体分组，并自动选择每组中的最优版本（优先 WOFF2 → WOFF → SFNT，同格式选体积最小）
3. **手动调整**：可以手动勾选/取消需要保留的文件
4. **选择目标格式**：在工具栏中选择要转换的目标格式
5. **转换并下载**：点击「转换并下载」，所有保留的字体将被转换为目标格式并打包成 ZIP 下载

## 技术说明

- 所有操作均在浏览器本地完成，字体文件不会上传到任何服务器
- 使用 opentype.js 解析字体元数据
- 使用 fflate 进行 zlib/Brotli 压缩解压
- 使用 JSZip 打包下载
