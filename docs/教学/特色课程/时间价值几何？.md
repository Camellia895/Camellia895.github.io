# 时间价值几何？

（第一课时）

## 1. 学情分析

二年级（约 7-8 岁）的学生正处于皮亚杰所说的“具体运算阶段”。他们对“金钱”有初步概念（知道能买零食、玩具），但对“寿命”和“未来”这种宏大概念缺乏直观感受。

我打算以买卖寿命作为讨论的主要问题。自己定审视自己的价值。


## 2. 教学目标

认识到寿命是有限切具体的。

能够审视当下的生活。

认识到寿命是不可逆的失去。（不是要求珍惜时间）

能够认识到当下生活中宝贵的东西。

## 3. 教学过程

导入：

提问：“要说什么东西是不可取代的、是珍贵无比的，应该就是每个人的生命。如果将人命换算成金额，大家觉得应该值多少钱呢？”

在黑板上画个小人，

“黑板上这个 不知其真面目的东西 非常有钱，而且它很憧憬能过一过人类的生活，希望有人可以将自己的人生卖给它。某天，你偶然从 这东西 面前经过时，它叫住你说： 喂，你可不可以把你剩下的人生全卖给我咧？ "

当然了，全部卖掉后大抵是会死掉的。

所以学生一定会拒绝拒绝它的条件。

不过 这东西 没有那么容易死心，又说：“那么，卖给我一半也行。要不要把你剩下的六十年寿命，折一半卖三十年给我呢？不能再少了。”

“那么，接下来才是真正的问题。想要一尝人类生活的『这东西』，该怎么替你的寿命估价呢？一年分的寿命该值多少钱？ 先说清楚，这个问题没有正确答案，我只是想听听大家的意见而已。请大家跟座位附近的同学讨论一下吧。”

“一年时间，你们觉得卖多少钱比较合适？是一千块？还是能买下一座大房子的钱？”

深入：

当孩子们给出了价格（比如一百万）

追问： “如果你卖掉了这一年，意味着你这一年里所有的周末、所有的生日蛋糕、所有和好朋友踢球的时间、甚至是你原本会发呆的那几个下午，全部都要归这个‘影子’。你拿到了钱，但你消失了。这一年，你还觉得值一百万吗？”
