# 学习日志-李崇建



## 摘录

> 当我聆听依莲的故事，以「好奇」介入她的叙事，了解更多她的内在，她的状态是如何形成。我亦时刻觉察自己，是否感觉任何不舒服？是否能接纳她所有的故事？这是从关怀自己为起点，才能平静的对依莲好奇、接纳与爱。
> 
> 作为一个陪伴者，我深知自己需要觉察，不被她卷入故事漩涡。她的故事会挑起陪伴者的情绪，一旦被挑起来而未觉察，自己也会被「旧意识」掌握，就会陷入拯救、保护、说教与指导的回圈，那就是陪伴者未守护自己的心。

在听别人将自己的事情的时候，需要避免陷入说教，指导的回旋。如果可以，应当是她自己提出的方法。

1.好奇很重要

2.念名字 且是念两个字 名字出现在句首 并和后续的语言加上停顿

3.停顿 用于给孩子思考的时间，以及在语言上带有承接感，孩子更愿意理你。

4.从具体的，孩子感情兴趣的事情，生活中的事物切入

5.如果想不到怎么回答的话，停顿，并重复上一句话的句尾 这有助于核对内容，看起来在倾听，也让自己停顿（不要太刻意，否则是敷衍）

6.精简叙述 先呼唤名字，不要打断孩子或者让说重点 帮他总结一下，大抵是三句话

7.将"为什么"换句话说。因为为什么被用烂了，导致看起来敷衍且不舒服。

换成 还好吗，发生什么事情了，怎么了，我很好奇，怎么办 

这些提问方法得混着用。善用怎么办、你觉得怎么办才好。这在让人负责，教育规则时候很有用。
