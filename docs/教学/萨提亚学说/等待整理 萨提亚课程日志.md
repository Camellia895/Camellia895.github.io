# 导师的萨提亚课程日志

> 导师开会的时候给我们上课，关于萨提亚的。尽管理论本身已经知道许多了，但是再次听到还是感觉颇有收益。 
> 
> 课上以联想词的方式记录笔记，趁着还有记忆，便开始解压开来。

## 总述

萨提亚的人本的，是相信人性本善的理论。

要相信时间的力量，人或许不会立刻改变，埋下种子即可

问题不是问题，如何应对才是问题。这是应对姿态的前言。

一样的青春期（生物状况）为什么每个人的行为程度不一样？ 这和过去的经历有关。

夸和捧 有着带着目的，带有不是基于事实的感觉。要换成认可，理解。

最深做到渴望层即可，剩下的自我层是留给他自己点亮的。

## 应对姿态

指责型

不要羞辱到人格。例如，你就是懒，拖拖拉拉。

孩子不一定喜欢指责的姿态对待人。

在观点，感受上不要花太多时间。而要看到学生的渴望。

只分析行为对错，一定会有人委屈。看到学生的渴望层，事情才能平息。

生气是表面，下面可能是委屈。

被指责对待的孩子，会逃避，会感到委屈，会生气，（这时候逃避不是问题，是解决策略）

讨好

区分出规矩和可选的，总是讨好会导致边界没有区分好

>晚饭吃什么？ 可乐
>
>晚饭吃什么，面，炒饭，米饭？面。

要说好边界。没有说出自己的边界所在，会导致被踩到边界。

这是种没有看到自己姿态。

超理智

冷漠，高高在上。感受上接近指责型，也能感觉自己被骂了。

拿大道理说事情。

孩子与之回应则是：关你什么事情

它没有看到活生生的人。
