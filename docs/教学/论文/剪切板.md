4. 萨提亚模式视角下的“日常行为偏差”界定

基于此，本研究尝试引入萨提亚的心理学视角，对小学生“日常行为偏差”进行重新界定，并分类。

在萨提亚模式看来，并不存在绝对的“坏行为”。萨提亚指出，个体的外显行为是内在心理冰山的表层流露。所谓的“行为偏差”，本质上是个体在压力情境下，因低自尊和内在不平衡而采取的“功能不良的生存应对姿态” [4][5]。

根据萨提亚的沟通理论，人们的应对姿态主要包括指责、讨好、打岔、超理智以及一致性五种。其中，“一致性”是身心和谐的健康目标，而其余四种则是失衡的表现。因此，本研究将“小学生日常行为偏差”界定为：小学生在学校生活中表现出的、偏离一致性状态的防御性应对行为。
也就是：

指责型行为：对应外显的攻击、破坏纪律与对抗行为；
讨好型：对应畏缩、过度顺从及缺乏主见的依赖行为；
打岔型：对应学业上的拖延、分心、逃避及注意力涣散；
超理智型：对应情感上的冷漠、隔离及缺乏同理心。
