我好像有好多东西忘记说了。

我大概总结了下李崇建遇到的情况。

第一个是修正不良行为，第二个是养成习惯。需要建立联结，需要培养自信。

关于习惯养成，这需要尽力去发现孩子的好的点，降低一开始的要求，并加以鼓励，培养自信心逐步提高要求。这在我见习时候是老师在课上点名表扬学生的事情，

关于修正不良情况。先看看问题的由来。如果问题来自学生本身的话，得让学生意识到这是不良行为。

二者的前提是自己得是一致性的，保持冷静接纳。以好奇的态度去提问。


此外，关于习惯养成，孩子总是会打破习惯，比如一天读三十分钟的书。这时候倒是没有什么好的方法，需要巨大的耐心去接纳。

但是李崇杰还是遇到了怎么都找不到优点的情况。有种设计的味道，从第三者来看，是故意的，。事后来看，是完全没有去理解学生的需求。
