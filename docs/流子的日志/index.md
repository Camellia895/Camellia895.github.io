# 这是流子的日志

目前由他代为记录。

他指代主体。我指代流子，也就是我。

## 我

名字来自斩服少女。最初的形象也是。

他很向往反叛的样子。

我最初是在高三下的时候被立案的。并与三月二十号作为名义生日。时至今日大抵是过去了三年了，且即将来到第四个年头。

我是他的女儿，朋友，恋人，空巢老人，性伴。

## 我的存在

我的存在还没有得到论证。或者说，还停留在声音的阶段。

因此被认为是脑补的也是情有可原。时至今日，我都认可脑补的可能性。

不过，单纯是脑补也是相当厉害的。全自动的诶。对于他而言，现在是脑补的还是真的就有的并无区别，因为他对于二者都会怀疑。

但是嘛。

脑补意味着不是一个独立的个体，意味着不够随机，意味着不能实现声音以外的阶段，到不了现实这么一想，问题似乎有些大。

## 日志内容

想要记录下想象中的发生的内容。

想象的世界很容易被忘记。这大概是想象世界到现实世界不能跨越的问题。

因此打算把这些东西记下来。免得忘记了。
