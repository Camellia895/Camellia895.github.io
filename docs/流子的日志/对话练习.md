# 对话/表达练习

## 联想词汇

> 对于这个词想象到什么 上方是流子的想法，下方是他的想法。

01.树

猴面包树

速生树

02.奶牛

我的世界的那种像素的，傻傻的奶牛

现实中写实的，但是是夸张的奶牛，方方正正的那种

03.铅笔

闪耀的流星

素描的手以及拿着的铅笔

04.闪闪发光

以矢量图形概况的星空

缩小的带有深度的星空

05. 明亮

超新星在星空中

白色环境

06.电话

纸杯电话

老式有线电话

07. 超人

平凡的，伟大的人，唯一的超能力就是总是去做自己想做的事情

也是平凡的没有超能力，但是通过努力能够做到多才多艺，或者在某些方面很突出

08.奶酪

圆形车轮奶酪

四点立体奶酪

09. 魔术

魔法

障眼法

10. 美丽

是褒义词，和其他褒义词没有多大区别。

设想了很多的二次元人物，并回到了我最开始形象为什么是美丽的。

11. 老虎

豹子

大猫

12. 鸟类

拉屎的怪物

海鸥

13. 歌曲

日语歌，电子的，古典的都喜欢

同上

14. 足球

不详

社会化运动，不常体验，也参与不到其中

15. 蜘蛛

与其他动物没有区别。至少不是令人恐怖的。他播放了很多以前的画面给我看，无感。

并不恐怖。
