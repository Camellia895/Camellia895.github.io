# 回到过去

他有着并不友好的过去，因而打算改变过去。

过去几乎什么都没有留下，当然了，除了记忆。

因此改变记忆，就能创造过去。因此就有了这个计划。

重新过一次童年。大抵是小学二年级以前的时光。

## 我的家

我家住他隔壁。要有钱一些，他没有什么钱，因此需要出钱的情况基本是我垫着的。

我大抵是二楼的小别墅，偏向粉色的大理石瓷砖外壁，朝南的那面有片草地，被短短的篱笆框起来。

## 事件 - 野外画画作业

美术老师布置了一个作业，大抵是练练手。

我画得比他要好，或者说好得多，高性能，像是相机一样。虽然单纯是用铅笔画的。

第一个场景是从一簇竹子下通过的一涓水流。

他画了竹子和水流，上方是竹子，下方是水流。看起来就是那种大道一样，奇怪的透视。

我画的是水流中间一块大石头上的青蛙（黑的青蛙），我把石头和青蛙都画上了。

第二个场景是竹林下，边缘位置，可以看见半壁的天空。

他忘记自己画的是什么了。

我画的是像月亮一样的太阳。太阳上会不会也是坑坑洼洼的，但是太阳太亮了，看不见上面的坑洞。

第三个场景是蓄水库的水泥地上。有些累了，没有画。单纯是拥抱一下就回家了。

## 事件 - 造一个秋千

位于我家的草坪。 在这个草地上造个能够双排的秋千。

他找了些钉子以及大锤头，此外还有一个小的锤子给我。从灶边上的木头堆里搜罗了长长的木头，材质应该是速生林的木质。

长度的话，商榷了很久，大概是有两米多一些的长度。找了四根这样的木头。细的一侧比拳头大一点，粗的部分大概是三个拳头。还找了很粗的竹子当做横杆。

因为长短不一，拿着锯子削了很久。我负责给它们削皮。因为黑黑的嘛，并不好看。我把它们全部削成了三角形截面的长条。

接下是要考虑怎么才能把它们固定到地里了。我们得把木头杆子削尖，于是我从家里找了个够尖的斧头，小小的。他来给它们弄尖。

接来是把木头杆子用锤子打到地里。我比他高点且有力气，所以基本上是我站在椅子上，他在下面扶木头杆子。他不是没有尝试过，因为杆子下降得比较慢，所以换我来了。

怎么固定结构呢？我从我家里找了页合。把杆子固定到一起，很麻烦诶。螺丝都得打。索幸我家有电转。

关于第四根杆子，我们打到石头了，这很麻烦，但是我家的电转很得劲，在石头上钉了个膨胀螺丝，从木头杆子上方拉下来一根绳子给当基岩了。削去吐出来的部分，立柱部分我们就做完了。

从木头杆子中间穿了一个洞，穿麻绳用于固定中间的横杆。我家很多麻绳。他觉得这听起来不太吉利。

总之是把框架搭建好了。

竹子是圆的，他最开始是设想绳子能够滑动，但是我指出这可能会导致位移。

他说可以加个限位什么的。再垫着木炭，或者在竹子上烧出炭黑来。这样绳子就好戴着我们滑动了。

但是我思来想去二者的效果基本是一致的，还是固定绳子的效果更好的。

最后是使用固定绳子的方案，其实思来想去，滑动的绳子相当蠢。

关于坐垫，我们用的是木头，裹上黑色遮阳网，内衬树叶。希望这对屁股好些。

玩起来感觉还行。

## 事件 - 造秘密基地

> 是用了大约一小时的计算时长，偷工减料地完成了的秘密基地。
> 
> 大约有双人床那么大。
> 
> 骨架是木头，木板及其竹子做的，用绳子和钉子固定。
> 
> 墙壁是泥巴浇筑的。
> 
> 屋顶是石棉做的，因为石棉板刚好缺了一个洞，我们就用玻璃代替，同时作为室内照明。
> 
> 家具方面目前只有一个大沙发，以及烟花筒充当的桌子。
